# Nassau Candy Optimization Dashboard

AI-powered supply chain optimization dashboard built using Python and Streamlit.

## Features

* Factory recommendation system
* Shipping optimization
* Interactive charts
* KPI dashboard
* Factory location map
* Download filtered data
* AI-based analytics

## Technologies Used

* Python
* Pandas
* Streamlit
* Matplotlib
* Plotly
* Scikit-Learn

## Project Structure

* app.py
* data/
* models/
* reports/
* requirements.txt

## Run Locally

```bash
streamlit run app.py
```
